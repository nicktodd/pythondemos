# Classes Lab Solutions

This folder contains complete solutions for the Python Classes and Objects lab.

## Files

- `basic_account.py` - Complete basic BankAccount class
- `account_with_statics.py` - BankAccount with static and class methods
- `account_operators.py` - BankAccount with operator overloading
- `account_strings.py` - BankAccount with string representations
- `savings_account.py` - SavingsAccount with inheritance
- `premium_account.py` - PremiumAccount with multiple inheritance

## Usage

Use these files to compare with your implementations or to see working examples.

Each file includes test code that demonstrates the functionality.
